隱私政策

最後更新：2026-07-21

本政策說明 hermes-pay 如何處理您的個人資料。

1. 我們收集的資料

  • 付款資料：付款由 Ko-fi、Stripe、Gumroad 等第三方平台處理。我們不會收集或儲存您的完整信用卡號。我們僅可能收到平台提供之交易確認、電子郵件與訂單資訊，用於交付服務。
  • 使用分析：本站可能使用 Google Analytics 透過 Cookie 收集匿名瀏覽統計。
  • 聯絡資訊：當您主動來信或於 Ko-fi 留言時所提供之資料。

2. 使用目的與法律依據

我們僅為履行合約（交付您購買的服務）、提供客戶支援與改善服務之目的處理資料。分析類 Cookie 之使用以您的同意為依據。

3. 資料分享

我們不會出售您的個人資料。資料僅與為提供服務所必需之第三方（付款平台、GitHub、Google）分享。

4. 您的權利

您有權要求查閱、更正或刪除我們持有之您的個人資料。如需行使權利，請透過下方方式聯絡我們，我們將於合理期間內回覆。

5. Cookie

您可透過瀏覽器設定停用 Cookie。第三方付款平台之資料處理另受其各自隱私政策規範。

6. 聯絡方式

隱私相關詢問請透過 Ko-fi 或 GitHub 與我們聯絡。
